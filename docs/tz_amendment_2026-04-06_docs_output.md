Отказываемся от итоговой Google Sheet в пользу текстового файла.

В отчетную папку ассистент (человек) по образцу помещает два заранее подготовленных пустых файла со следующими именами:

* Сводная таблица по магистерским проектам, (открытая ссылка, ландшафтное представление);
* Детальная таблица по магистерским проектам, (открытая ссылка, книжное представление))

Далее ассистент запускает бота и передает ему 3 параметра:

* адрес файла, содержащий имена отчетов магистрантов (продумать автоматизацию передачи ссылки);
* адрес ссылки вновь созданного файла Сводной таблицы по магистерским проектам;
* адрес ссылки вновь созданного файла Детальной таблицы по магистерским проектам.
