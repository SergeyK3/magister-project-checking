## Что уже сделано в коде

- **`/start`:** если есть строка в «Регистрация» → продолжение анкеты; иначе лист «Администраторы» (active) → приветствие админа; иначе лист **«научрук»** → приветствие научрука; иначе **инлайн-кнопки** (магистрант / научрук / админ).
- **Магистрант:** подменю «Новая регистрация» или «Уже есть запись (по ФИО)» → второе ведёт к старой привязке к **«Регистрация»**.
- **Админ / научрук впервые:** ФИО по своему листу → да/нет → `attach_telegram_to_row` на этот лист (см. `CLAIM_ASK_FIO` / `CLAIM_CONFIRM`, callback `^start:`).
- **`/help`:** приоритет админ → научрук → магистрант; текст научрука — MVP (`HELP_REPLY_TEXT_SUPERVISOR`).

В том же коммите на ветке есть и **другая** работа (spravka, project snapshot JSON, `telegram_force_ipv4`, правки row_check / project_card / config) — передатка ниже про **роли и научрук**; остальное смотри сообщение коммита `git show cc2825e --stat`.
