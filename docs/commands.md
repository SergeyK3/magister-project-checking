# Commands

## Общие принципы

Пользователю показываются только русскоязычные команды.

Латинские команды являются alias-командами.

---

## Общие команды

| Команда    | Alias    | Назначение      |
| ---------- | -------- | --------------- |
| /старт     | /start   | запуск системы  |
| /справка   | /spravka | запуск проверки |
| /выход     | /exit    | выход           |
| /о_проекте | /about   | информация      |

---

## Команды магистранта

| Команда  | Назначение       |
| -------- | ---------------- |
| /справка | проверка проекта |

---

## Команды научрука

| Команда          | Назначение           |
| ---------------- | -------------------- |
| /справка Фамилия | проверка магистранта |
| /список          | список магистрантов  |

---

## Команды админа

| Команда          | Назначение            |
| ---------------- | --------------------- |
| /справка Фамилия | проверка проекта      |
| /список          | список магистрантов   |
| /ошибка          | замечание магистранту |
