# Руководство научного руководителя. V3

## Работа с ботом

Все действия выполняются в личном чате с ботом.

---

# Первый запуск

Команда:

```text
/start
```

запускает:

* идентификацию;
* определение роли;
* привязку Telegram ID.

---

# Основные возможности

Научный руководитель может:

* контролировать проекты магистрантов;
* получать информацию о состоянии проектов;
* просматривать списки магистрантов.

---

# Команды

| Команда            | Назначение           |
| ------------------ | -------------------- |
| `/start`           | запуск системы       |
| `/справка Фамилия` | проверка проекта     |
| `/список`          | список магистрантов  |
| `/выход`           | завершение диалога   |
| `/о_проекте`       | информация о проекте |

---

# Команда `/справка`

Пример:

```text
/справка Иванов
```

Команда запускает:

* повторную проверку проекта;
* анализ ссылок;
* формирование справки.

---

# Команда `/список`

Показывает:

* список магистрантов;
* статус регистрации;
* контактные данные.

---

# Работа с Google Таблицей

Основные вкладки:

* Магистранты
* Регистрация

---

# Важно

Если магистрант не зарегистрировался:

1. свяжитесь с ним;
2. попросите выполнить:

```text
/start
```
